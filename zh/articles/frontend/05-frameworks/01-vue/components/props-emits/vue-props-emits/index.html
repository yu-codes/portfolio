Vue 3 Props & Emits Complete Guide: Core Component Communication

Yu 最後更新: 2025-03-01

Props and Emits are the foundation of Vue component communication. Props handle parent-to-child data flow, Emits handle child-to-parent event flow, together forming the cornerstone of Vue’s uni-directional data flow.

Core Concepts

Props Definition & Validation

// TypeScript definition (recommended)
const props = defineProps<{
  title: string
  count?: number
  items: string[]
}>()

// With defaults
const props = withDefaults(defineProps<{
  msg?: string
  count?: number
}>(), {
  msg: 'hello',
  count: 0
})

Emits Events

const emit = defineEmits<{
  change: [id: number]
  update: [value: string]
}>()

emit('change', 42)
emit('update', 'new value')
