Vue 3 Lifecycle Complete Visual Guide: Every Hook from Creation to Destruction

Yu 最後更新: 2025-03-01

Every Vue component goes through a series of initialization steps — setting up reactive data, compiling templates, mounting DOM, updating and destroying. Lifecycle hooks let you execute custom logic at these critical points.

Hook Usage Scenarios

import {
  onMounted, onUnmounted,
  onActivated, onDeactivated,
  onErrorCaptured
} from 'vue'

// Most common: execute after DOM mount
onMounted(() => {
  // Safely access DOM elements
  // Initialize third-party libs (Chart.js, maps, etc.)
  // Send initial API requests
})

// Cleanup: remove side effects on unmount
onUnmounted(() => {
  // Clear setInterval/setTimeout
  // Remove event listeners
  // Disconnect WebSocket
})

// Keep-Alive specific
onActivated(() => { /* Component activated */ })
onDeactivated(() => { /* Component deactivated */ })

// Error capture
onErrorCaptured((err, instance, info) => {
  console.error('Child component error:', err)
  return false // Stop propagation
})
