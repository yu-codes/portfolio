線性代數與統計學

機器學習的數學基石：線性代數與機率統計的核心觀念

機器學習的一切，都建立在數學之上。而線性代數與統計學，正是這棟大廈的地基。你不需要成為數學家，但理解核心觀念，能讓你在面對模型行為時，不再只是「調參」，而是真正理解背後的原理。

為什麼線性代數對 ML 如此重要？

機器學習中的資料，本質上就是向量與矩陣。一張圖片是像素矩陣，一段文字經過 embedding 後是向量，一批訓練資料就是一個大矩陣。

當你執行一次神經網路的前向傳播（forward pass），本質上就是在做矩陣乘法。理解矩陣運算，等於理解模型的計算核心。

向量：資料的基本單位

向量代表 n 維空間中的一個點。在 ML 中，每個資料樣本都可以表示成一個特徵向量。

  • 向量加法：特徵的組合
  • 內積（dot product）：衡量兩個向量的相似度，是 attention 機制的核心
  • 範數（norm）：衡量向量的「大小」，L1 範數（稀疏性）與 L2 範數（平滑性）在正則化中扮演關鍵角色

矩陣：資料的組織方式

矩陣可以看作多個向量的堆疊，也可以看作一個線性變換。

  • 矩陣乘法：神經網路每一層的運算核心
  • 轉置（transpose）：調整資料維度
  • 逆矩陣：在最小平方法等場景中求解參數
  • 行列式（determinant）：判斷變換是否可逆

矩陣分解：降維的利器

矩陣分解是許多 ML 演算法的基礎：

分解方法核心概念應用場景
特徵分解找出矩陣的主要方向PCA、譜聚類
SVD（奇異值分解）將矩陣拆成三個矩陣的乘積推薦系統、降維、資料壓縮
Cholesky 分解將正定矩陣分解為下三角矩陣高效求解線性系統

PCA（主成分分析） 就是特徵分解的直接應用——找出資料中方差最大的方向，將高維資料投影到低維空間，同時保留最多資訊。

統計學：理解資料的語言

如果線性代數是 ML 的「骨架」，統計學就是它的「感知系統」——讓你能從資料中萃取出有意義的資訊。

機率分佈：描述不確定性

資料的分佈特徵，直接影響你該選用什麼模型。

常見離散分佈：

  • 伯努利分佈：二元分類的基礎（是/否）
  • 多項分佈：多類別分類
  • 泊松分佈：描述事件發生的頻率

常見連續分佈：

  • 常態分佈（高斯分佈）：許多自然現象的近似，也是許多演算法的假設前提
  • 均勻分佈：隨機初始化權重時常用
  • Beta 分佈：貝葉斯推論中常用的先驗分佈

貝葉斯定理：從證據更新信念

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

貝葉斯定理是機率推論的核心。它告訴我們如何根據觀察到的證據（資料），更新我們對假設的信念。

在 ML 中，這就是後驗機率 = 似然度 × 先驗機率 / 邊際機率。樸素貝葉斯分類器、貝葉斯優化、變分推論，都建立在這個基礎上。

最大似然估計（MLE）

給定資料，找出讓資料出現機率最大的參數值。這是許多 ML 模型訓練的本質——最小化 loss function，往往等價於最大化似然函數。

假設檢定：判斷結果是否有意義

在評估模型效能時，統計假設檢定幫助你判斷：「這個模型真的比較好，還是只是隨機波動？」

  • p 值：觀察到的結果在虛無假設下出現的機率
  • 信賴區間：參數的可能範圍
  • A/B 測試：在生產環境中比較兩個模型的表現

關鍵心法

  1. 向量內積是衡量相似度的基礎，從餘弦相似度到 attention 機制都在用
  2. 矩陣分解是降維與特徵提取的核心工具
  3. 機率分佈的假設會直接影響模型的選擇與效能
  4. 貝葉斯思維讓你能系統性地從資料中學習

掌握這些概念，你就能看穿大多數 ML 演算法的本質，而不只是把它們當成黑箱使用。
