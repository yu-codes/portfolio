模型建構

用深度學習框架建構神經網路：從基礎架構到現代設計模式

建構一個神經網路不難，建構一個好的神經網路需要理解各種架構的設計原理和適用場景。

基礎神經網路架構

全連接網路（MLP）

最基礎的架構：每個神經元都與上一層的所有神經元相連。

組成要素：

  • 線性層：矩陣乘法 + 偏差
  • 啟動函數：引入非線性（ReLU、GELU 等）
  • 堆疊：多層線性 + 啟動的組合

MLP 的問題：參數量隨輸入維度線性增長。對於圖片（上萬維）或序列（長度可變），不實用。

卷積神經網路（CNN）

CNN 的核心直覺：局部連接 + 權重共享。

  • 卷積層：用一個小的過濾器（kernel）在輸入上滑動，提取局部特徵
  • 池化層：降低空間維度，增加平移不變性
  • 特徵階層：淺層學邊緣和紋理，深層學形狀和物體

經典架構演進：LeNet → AlexNet → VGG → ResNet → EfficientNet

循環神經網路（RNN）

處理序列資料的架構。每個時間步都有一個隱藏狀態，攜帶之前的資訊。

問題：長序列中的梯度消失。

LSTM（長短期記憶）：加入門控機制（輸入門、遺忘門、輸出門），能選擇性地記憶或遺忘資訊。

GRU：LSTM 的簡化版，參數更少，效能相當。

注意：RNN 在大多數場景下已被 Transformer 取代。

Transformer

2017 年以來最具影響力的架構。核心機制是 Self-Attention。

Self-Attention 的直覺：對於序列中的每個位置，計算它應該「注意」其他哪些位置。這讓模型能直接捕捉長距離依賴，不受序列長度限制（理論上）。

Transformer 的組成：

  • Multi-Head Attention：多組 attention 平行運算，捕捉不同面向的關係
  • Feed-Forward Network：位置級的非線性變換
  • Layer Normalization：穩定訓練
  • Residual Connection：確保梯度流通

現代設計模式

Residual Connection（殘差連接）

$y = f(x) + x$

讓網路學習「殘差」而非「完整映射」。這個看似簡單的技巧讓訓練非常深的網路成為可能（100+ 層）。

Normalization

  • Batch Normalization：在批次維度上正規化。CNN 的標配。
  • Layer Normalization：在特徵維度上正規化。Transformer 的標配。
  • Group Normalization：BatchNorm 和 LayerNorm 的折衷。

注意力機制

  • Self-Attention：序列內部的注意力
  • Cross-Attention：兩個序列之間的注意力（如 encoder-decoder）
  • Multi-Head Attention：多組 attention 捕捉不同關係
  • Flash Attention：計算效率優化版的 attention

位置編碼

Transformer 本身不理解位置。需要額外的位置資訊。

  • 正弦位置編碼：原始 Transformer 使用
  • 可學習位置編碼：BERT 使用
  • 旋轉位置編碼（RoPE）：LLaMA 等現代 LLM 使用
  • ALiBi：不用位置編碼，直接在 attention 分數上加偏差

架構選擇指南

任務推薦架構
圖片分類Vision Transformer / EfficientNet
物件偵測DETR / YOLOv8
文字分類BERT fine-tune
文字生成GPT 架構
序列到序列Encoder-Decoder Transformer
表格資料MLP / TabNet（但 GBDT 通常更好）

實務心法

  1. 大多數情況下不需要自己設計架構——用預訓練模型 + fine-tune
  2. Transformer 幾乎適用於所有模態——文字、圖片、音訊、時間序列
  3. ResNet 和 BERT 依然是很好的基準線
  4. 關注模型的計算效率——不是越大越好
