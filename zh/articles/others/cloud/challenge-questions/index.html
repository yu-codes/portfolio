解題技巧與常見誤區

  • 計算題：先寫公式再帶數字，避免粗心。
  • 推導題：分步驟寫清楚，標明假設。
  • 直覺題：用圖解、生活例子輔助說明。
  • 實作題：熟悉 Terraform、K8s YAML、CI/CD、雲端 API 等常用工具。
  • 常見誤區：混淆定義、忽略權限、過度依賴單一雲、缺乏自動化。
