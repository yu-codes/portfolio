📘 第一部分：Python 字串語法大全（str 操作技巧）

Python 的 str 是不可變資料型別（immutable），可進行大量內建轉換與處理。

🧱 建立與基本操作

s = "hello"
len(s)            # 長度
s[0]              # 索引取值
s[-1]             # 從後面取值
s[1:4]            # 切片，不包含尾端
s[::-1]           # 字串反轉
"e" in s          # 成員檢查

🔤 常見轉換與清理

s.upper()         # 轉大寫
s.lower()         # 轉小寫
s.strip()         # 去除頭尾空白
s.replace("a", "b") # 替換字元
s.split(" ")      # 拆字串
"-".join(list)    # 合併為字串
s.startswith("he")
s.endswith("lo")

🔎 查找與比較

s.find("l")        # 傳回第一個 l 的 index（找不到回 -1）
s.index("l")       # 與 find 類似，但找不到會報錯
s.count("l")       # 出現次數
s == "hello"       # 相等比較
s.isdigit()        # 是否為數字
s.isalpha()        # 是否為字母

✍️ 特殊處理技巧

# 快速建字串
"".join([chr(97 + i) for i in range(26)])

# f-string 格式化
name = "John"
f"Hello, {name}!"

2. 🧠 雙指針與滑動視窗技巧

  • 判斷是否為回文

  • 子字串匹配
def is_palindrome(s):
    l, r = 0, len(s)-1
    while l < r:
        if s[l] != s[r]: return False
        l += 1; r -= 1
    return True
