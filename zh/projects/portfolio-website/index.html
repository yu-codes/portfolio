個人作品集網站

使用 Hugo 和現代 CSS 構建的個人作品集。具有互動式思維導圖導航、深淺模式支持和響應式設計。

View on GitHub

概述

這個作品集網站作為我的個人技術知識庫和專案展示平台。使用 Hugo 靜態網站生成器構建，具有現代化、響應式設計和互動式路線圖導航系統。

主要功能

  • 互動式路線圖：XMind 風格的可折疊節點視覺化，用於瀏覽技術文章
  • 深淺模式：無縫主題切換，支援系統偏好設定檢測
  • 多語言支援：完整的英文和繁體中文支援
  • 響應式設計：行動優先的自適應佈局
