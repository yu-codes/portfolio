即時聊天系統

基於 WebSocket 的聊天服務器，具有用戶認證和消息持久化。

Node.js Redis MongoDB

概述

支援多個聊天室、私訊和訊息歷史的可擴展即時聊天應用程式。

功能

  • 即時訊息：基於 WebSocket 的即時通訊
  • 聊天室：支援公開和私人房間
  • 私訊：一對一對話
  • 訊息持久化：完整的聊天歷史與搜尋
  • 用戶在線狀態：線上/離線狀態追蹤
  • 輸入指示器：即時輸入狀態

技術架構

後端技術棧

  • 伺服器：Node.js + Express
  • WebSocket：Socket.IO
  • 快取：Redis（會話和在線狀態）
  • 資料庫：MongoDB（訊息儲存）

數據流

用戶 A                 伺服器                用戶 B
  │                     │                     │
  │──── 訊息 ──────────▶│                     │
  │                     │──── 廣播 ──────────▶│
  │                     │                     │
  │                     │◀─── 確認 ───────────│
  │◀─── 送達通知 ───────│                     │

效能

  • 每個實例支援 10,000+ 並發連接
  • 平均訊息延遲：< 50ms
  • 99.9% 正常運行時間 SLA

安全性

  • 基於 JWT 的認證
  • 傳輸中訊息加密
  • 防濫用速率限制
  • 輸入消毒
