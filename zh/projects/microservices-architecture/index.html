微服務架構

可擴展的後端系統，具有容器化微服務、API 網關和消息隊列。

View on GitHub

概述

展示構建可擴展、可維護後端系統最佳實踐的生產就緒微服務架構。

關鍵組件

API 網關

  • 速率限制
  • 認證/授權
  • 負載均衡

服務網格

  • 服務發現
