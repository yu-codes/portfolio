Overview

A comprehensive technical platform combining personal portfolio, learning hub, and project showcase. Built entirely with Hugo static site generation, featuring a sophisticated interactive roadmap visualization system inspired by XMind. Supports dual-language content (English & Traditional Chinese) and deploys seamlessly to GitHub Pages with zero backend requirements.

Key Features

  • XMind-Style Interactive Roadmap: Advanced SVG-based collapsible node visualization with auto-layout algorithm, smooth animations, and context-aware link rendering for navigating 100+ technical articles across 9 domains
  • Sophisticated Dark Mode: Persistent theme preference with system detection fallback, powered by CSS variables and localStorage
  • Full Bilingual Support: English & Traditional Chinese with separate content directories and auto-generated multi-language navigation
  • Mobile-First Responsive Design: Adaptive layouts for all device sizes with CSS Grid and Flexbox
  • Archive Timeline View: Chronological sorting of articles by publication date with filtering capabilities
  • SEO Optimized: Auto-generated sitemaps for both languages, structured metadata, and semantic HTML

Architecture

Content Organization

  • Articles: Organized across 9 domains (Backend, Cloud, AI, etc.) with 3 engineering role perspectives
  • Projects: 11 showcase projects with detailed descriptions and links
  • Resume: Multi-format resume section
  • Archives: Timeline view of all content sorted by update date

CSS Architecture (Modular & Maintainable)

  • base.css - CSS reset, design tokens (variables), typography foundation
  • layout.css - CSS Grid system, header/footer, sidebar navigation
  • components.css - Reusable UI components (buttons, cards, code blocks)
  • pages/*.css - Page-specific overrides and optimizations

JavaScript Components

  • theme.js - Dark/light mode toggle with persistence
  • roadmap.js - 660-line XMind renderer with dynamic node layout and animations
