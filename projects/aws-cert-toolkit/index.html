Overview

A comprehensive AWS certification learning platform and production toolkit designed for developers and DevOps engineers. This project integrates certification study guides, interactive Jupyter notebooks with practical code examples, and reusable AWS utilities for rapid infrastructure deployment.

Key Features

  • Certification Learning: Complete study guides for Solutions Architect Associate, Developer Associate, and SysOps Administrator certifications
  • Interactive Tutorials: Jupyter notebooks with hands-on practice for EC2, S3, DynamoDB, Lambda, and CloudFormation
  • High-Level AWS Managers: Wrapper utilities for common AWS operations (S3, Lambda, IAM, CloudFormation)
  • Infrastructure as Code: CloudFormation and Terraform templates for rapid deployment
  • Docker Support: Container-based development environment with Jupyter Lab integration
  • Production-Ready: Built with best practices including logging, validation, and configuration management

Project Structure

  • certification/ - Structured study guides by certification level and domain topics
  • tutorials/ - Interactive Jupyter notebooks with practical code examples
  • toolkit/managers/ - High-level AWS operation wrappers (S3, Lambda, IAM, CloudFormation)
  • toolkit/templates/ - Production-ready IaC templates (CloudFormation and Terraform)
  • utils/ - Utility functions for logging, validation, and AWS operations
  • config/ - AWS session and credential management

Architecture

The toolkit follows a modular architecture:

  • Manager Layer: High-level abstractions over AWS services for common operations
  • Template Layer: Pre-built IaC templates for infrastructure deployment
  • Utility Layer: Shared functions for logging, validation, and AWS interactions
  • Configuration Layer: Centralized AWS credential and session management
  • Learning Layer: Study guides and interactive tutorials for skill development
