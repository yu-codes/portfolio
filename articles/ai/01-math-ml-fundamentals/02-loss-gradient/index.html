損失函數與梯度下降

理解模型如何學習：損失函數的設計與梯度下降的優化原理

機器學習模型的訓練，本質上就是一個優化問題：找到一組參數，讓損失函數的值最小。損失函數告訴你「模型有多差」，梯度下降則告訴你「該往哪個方向調整參數」。

損失函數：衡量模型表現的量尺

損失函數（Loss Function）把模型預測值與真實值之間的差距，壓縮成一個數字。這個數字越小，代表模型表現越好。

回歸任務的損失函數

損失函數公式概念特性
MSE（均方誤差）預測誤差的平方平均對離群值敏感，梯度隨誤差增大
MAE（平均絕對誤差）預測誤差的絕對值平均對離群值較穩健，但在零點不可微
Huber Loss小誤差用 MSE，大誤差用 MAE結合兩者優點

MSE 是最常見的回歸損失，但如果你的資料有很多離群值，Huber Loss 是更好的選擇。

分類任務的損失函數

二元交叉熵（Binary Cross-Entropy） 是二元分類的標準損失。它的本質是衡量預測的機率分佈與真實分佈之間的「距離」。

分類交叉熵（Categorical Cross-Entropy） 則用於多類別分類，概念完全相同，只是擴展到多個類別。

交叉熵的巧妙之處在於：當模型非常確定但答錯時，損失會極大（趨近無限大），這迫使模型不要過度自信。

損失函數的選擇策略

選擇損失函數時，你需要考慮：

  1. 任務類型：回歸用 MSE/MAE，分類用交叉熵
  2. 資料特性：有離群值嗎？類別不平衡嗎？
  3. 你在意什麼：對大誤差的容忍度、對確定性的要求
  4. 可微性：損失函數需要可微才能計算梯度

梯度下降：朝最佳解前進

有了損失函數，下一步就是找到讓損失最小的參數。梯度下降是最核心的方法。

直覺理解

想像你站在一座山上，被濃霧包圍看不見全貌。你唯一能做的，就是感受腳下地面的斜度，然後朝最陡的下坡方向走一步。重複這個過程，最終你會走到谷底。

這就是梯度下降：

  • 梯度：損失函數在當前位置的斜率方向（指向上坡最快的方向）
  • 學習率：每一步走多遠
  • 更新規則：參數 = 參數 - 學習率 × 梯度

學習率的藝術

學習率是最關鍵的超參數之一：

  • 太大：步伐太大，可能跳過最佳解，甚至發散
  • 太小：收斂太慢，可能卡在局部最小值
  • 恰到好處：需要實驗找到平衡點

實務上常用學習率排程（Learning Rate Schedule）：

  • 餘弦退火（Cosine Annealing）：像鐘擺一樣逐漸減速
  • 暖啟動（Warmup）：一開始用小學習率，逐漸增大再減小
  • 指數衰減：每隔固定步數乘以一個衰減係數

梯度下降的變體

方法核心概念適用場景
BGD（批次梯度下降）用全部資料計算梯度資料量小時
SGD（隨機梯度下降）每次只用一筆資料大規模資料
Mini-batch GD用一小批資料計算梯度最常用的實務選擇

進階優化器

SGD 的問題在於它對所有參數使用相同的學習率，且容易被噪聲影響。進階優化器解決了這些問題：

  • Momentum（動量）：記住之前的移動方向，減少震盪。就像球從山上滾下來，會累積速度。
  • RMSprop：對每個參數自適應調整學習率。頻繁更新的參數用較小學習率。
  • Adam：結合 Momentum 和 RMSprop 的優點，是目前最廣泛使用的優化器。
  • AdamW：在 Adam 基礎上修正了權重衰減的實作，是目前訓練 Transformer 的首選。

反向傳播：梯度計算的引擎

在多層神經網路中，損失函數對參數的梯度要怎麼算？答案是反向傳播（Backpropagation）。

反向傳播的核心就是鏈式法則（Chain Rule）：

如果 $y = f(g(x))$，那麼 $\frac{dy}{dx} = \frac{dy}{dg} \cdot \frac{dg}{dx}$

神經網路就是一連串函數的組合。反向傳播從輸出層開始，逐層往回計算每個參數的梯度，效率非常高。

梯度消失與梯度爆炸

在很深的網路中，梯度可能會：

  • 消失：梯度越傳越小，前面的層幾乎學不到東西（常見於 Sigmoid 啟動函數）
  • 爆炸：梯度越傳越大，參數更新失控

解決方案包括：

  • 使用 ReLU 等不會壓縮梯度的啟動函數
  • 殘差連接（Residual Connection）：讓梯度有「捷徑」直接傳遞
  • 批次正規化（Batch Normalization）：穩定每層的輸入分佈
  • 梯度裁剪（Gradient Clipping）：限制梯度的最大值

實務心法

  1. 先用 Adam 開始，如果效果不好再嘗試 SGD + Momentum
  2. 學習率是最值得花時間調的超參數，建議用學習率搜尋（LR Finder）
  3. 觀察 loss 曲線：平穩下降是好的，劇烈震盪代表學習率太大
  4. 梯度裁剪在 RNN 和 Transformer 訓練中幾乎是標配
