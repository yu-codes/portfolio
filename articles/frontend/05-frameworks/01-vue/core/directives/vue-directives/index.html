Vue 3 Directives Complete Guide: v-model, v-if/v-for & Custom Directives

Yu Last updated: 2025-03-01

Directives are special attributes prefixed with v- in Vue templates that bind reactive behavior to the DOM. This article covers all built-in directives and custom directive development.

Core Concepts

v-model Two-way Binding

<!-- Basic usage -->
<input v-model="message">

<!-- Equivalent to -->
<input :value="message" @input="message = $event.target.value">

v-model Modifiers

<input v-model.lazy="msg">     <!-- change event instead of input -->
<input v-model.number="age">   <!-- Auto-convert to number -->
<input v-model.trim="msg">     <!-- Auto-trim whitespace -->

v-model on Components

<!-- Vue 3 supports multiple v-model -->
<UserForm
  v-model:first-name="first"
  v-model:last-name="last"
/>

// UserForm.vue
defineProps<{ firstName: string; lastName: string }>()
defineEmits<{
  'update:firstName': [value: string]
  'update:lastName': [value: string]
}>()

Custom Directives

// Global registration
app.directive('focus', {
  mounted(el) {
    el.focus()
  }
})

// Local registration (in <script setup>)
const vFocus = {
  mounted: (el: HTMLElement) => el.focus()
}

Use Cases

  • Auto-focus: v-focus directive to auto-focus input fields
  • Permission control: v-permission directive to hide elements by role
  • Infinite scroll: v-infinite-scroll to detect scroll to bottom
  • Tooltip: v-tooltip directive for hover tips
