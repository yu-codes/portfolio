Yu Last updated: 2025-03-01

Core Concepts

Text Interpolation

<span>{{ message }}</span>
<span>{{ count + 1 }}</span>
<span>{{ ok ? 'YES' : 'NO' }}</span>
<span>{{ message.split('').reverse().join('') }}</span>

Double curly braces only support expressions, not statements (if/for, etc.).

Raw HTML

<span v-html="rawHtml"></span>

⚠️ Warning: v-html has XSS risks – only use with trusted content.

Attribute Binding

<!-- Dynamic attributes -->
<img :src="imageUrl" :alt="desc">

<!-- Dynamic class -->
<div :class="{ active: isActive, error: hasError }"></div>
<div :class="[baseClass, isActive ? 'active' : '']"></div>

<!-- Dynamic style -->
<div :style="{ color: textColor, fontSize: size + 'px' }"></div>

<!-- Bind multiple attributes -->
<div v-bind="objectOfAttrs"></div>

Conditional Rendering

<div v-if="type === 'A'">A</div>
<div v-else-if="type === 'B'">B</div>
<div v-else>Other</div>

<!-- v-show: CSS display toggle, DOM stays -->
<div v-show="isVisible">Always in DOM</div>

v-if vs v-show:

  • v-if: Real destroy/create, higher toggle cost
  • v-show: CSS toggle, higher initial render but fast toggle

List Rendering

<!-- Array -->
<li v-for="(item, index) in items" :key="item.id">
  {{ index }} - {{ item.name }}
</li>

<!-- Object -->
<li v-for="(value, key, index) in object" :key="key">
  {{ key }}: {{ value }}
</li>

<!-- Range -->
<span v-for="n in 10" :key="n">{{ n }}</span>

⚠️ Always provide :key, and avoid using index as key.

Event Handling

<!-- Method handler -->
<button @click="increment">+1</button>

<!-- Inline handler -->
<button @click="count++">+1</button>

<!-- Pass event object -->
<button @click="handler($event)">Click</button>

<!-- Modifiers -->
<form @submit.prevent="onSubmit">
<input @keyup.enter="submit">
<button @click.stop="doThis">
<div @scroll.passive="onScroll">

Use Cases

  • Form validation: computed + v-if to control error message display
  • Dynamic tables: v-for rendering rows, :class for selected row highlighting
  • Permission control: v-if to show/hide features based on user roles
